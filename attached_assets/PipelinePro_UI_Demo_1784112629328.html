Attribute Mapping — Edit Rule
Four-column progressive rule builder. Each column appears after the previous one is filled.
Rules are evaluated left to right for every product during the Enrich stage. Non-AI rules always take priority over AI extraction.

When to apply this rule
Rule applies only when product is in this Sunsky category
Rule summary
For the WooCommerce attribute Color:
Use AI extraction with instruction: "Find the color of the product..."
Apply: Always
Priority note
Non-AI rules (fixed value, Sunsky attribute) always take priority over AI extraction for the same attribute. AI runs only if no non-AI rule matched.
