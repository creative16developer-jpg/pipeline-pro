Project Roadmap

4 MILESTONES

Milestones are structured around working functionality, not technical phases. Each milestone delivers something you can validate before we move forward.

✓ AGREED WORKING TERMS
Functionality-first milestones
Each milestone = working, testable output
AI via your API endpoint
No AI code shared — calls your endpoint only
Shared Git repository
All work delivered via shared private repo
M1
End-to-End Pipeline — Small Batch
10–20 real products live on WooCommerce as drafts. No AI content yet.
IN SCOPE ~30 hrs
Sunsky API connected with real credentials (auth + pagination)
Product images downloaded to local storage
WooCommerce draft products created (title, SKU, price, stock, images)
Dashboard shows job status + product list with WooCommerce IDs
Basic description from Sunsky raw data (no AI)
Client reviews drafts in WooCommerce admin ← validation point
Goal: Both parties validate the full pipeline on real data before scaling. Any issues are caught early.
M2
Full Processing Pipeline + AI Content
50–100 products with AI-enriched content and processed images.
IN SCOPE ~40 hrs
AI endpoint integration (descriptions, SEO metadata, tags, alt text)
Image pipeline: download → compress → watermark → WebP conversion
Batch processing with error handling and retry logic
Resume capability — skips already-processed products
Duplicate detection by SKU
Full batch of 50–100 products uploaded as WooCommerce drafts
Requires from you: Your AI API endpoint URL + auth token, and the existing Python image processing script.
M3
Dashboard, Multi-Store & Daily Sync
Production-ready management UI with automated ongoing sync.
IN SCOPE ~40 hrs
Polished dashboard (import history, per-product status, job logs)
Multi-store support (upload to different WooCommerce sites)
Category + attribute sync from WooCommerce
Daily price and stock sync via cron (by SKU)
CSV / XLS export of all product data
Simulation mode + duplicate detection + resume system
Goal: Fully automated, hands-off daily sync. Products stay up to date without manual intervention.
M4
Testing, Performance & Handover
Stable, documented system ready for your VPS and long-term use.
IN SCOPE ~22 hrs
Integration tests — Sunsky and WooCommerce endpoints
End-to-end flow tests (fetch → process → upload)
Performance testing (500+ products, rate limit handling)
Deployment to your Linux VPS with cron setup
Bug fixes and final polish
Full handover via Git — documented, clean history
Requires from you: Access to your Linux VPS and hosting environment for final deployment.
